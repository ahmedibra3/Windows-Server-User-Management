# Windows Server 2019 User Management Project

## Overview
This project involves managing user permissions and access controls in a Windows Server 2019 environment. The goal was to ensure secure and efficient user management while maintaining system integrity.

## Technologies Used
- Windows Server 2019
- Active Directory
- DNS
- FTP Server

## How to Use
To replicate this project, set up a Windows Server 2019 environment and configure user permissions and access controls as described.
